## AI Categorization: The Core Feature

### Testing Phase

I tested 2 models on 10K real expenses:

| Model | Accuracy | Latency | Cost/1K |
|-------|----------|---------|---------|
| Google Gemini 1.5 Flash | 94% | 450ms | $0.15 |
| OpenAI GPT-4o-mini | 96% | 380ms | $0.45 |

**Decision:** Gemini primary (cheaper), OpenAI fallback (better accuracy).

### Implementation

```python
from google import generativeai as genai
import openai

async def categorize_expense(text: str, categories: list[str]) -> str:
    """Categorize expense using AI"""

    # Try Gemini first
    try:
        prompt = f"""
        Categorize this expense: "{text}"
        Available categories: {', '.join(categories)}
        Return only the category name.
        """

        model = genai.GenerativeModel('gemini-1.5-flash')
        response = await model.generate_content_async(prompt)
        category = response.text.strip()

        if category in categories:
            return category

    except Exception as e:
        logger.warning(f"Gemini failed: {e}")

    # Fallback to OpenAI
    try:
        response = await openai.ChatCompletion.acreate(
            model="gpt-4o-mini",
            messages=[{"role": "user", "content": prompt}]
        )
        return response.choices[0].message.content.strip()

    except Exception as e:
        logger.error(f"Both AI providers failed: {e}")
        return "Other"  # Default category
```

**Key learnings:**
- Always have a fallback (99.9% uptime)
- Cache common patterns (faster, cheaper)
- Timeout protection (max 5s)

## PDF Reports with Playwright

Initially used WeasyPrint, but had issues:
- ‚ùå Poor support for complex CSS
- ‚ùå Rendering bugs with charts
- ‚ùå Limited font support

Switched to Playwright:
```python
from playwright.async_api import async_playwright

async def generate_pdf_report(html_content: str) -> bytes:
    async with async_playwright() as p:
        browser = await p.chromium.launch()
        page = await browser.new_page()
        await page.set_content(html_content)

        pdf = await page.pdf(
            format='A4',
            print_background=True,
            margin={'top': '1cm', 'bottom': '1cm'}
        )
        await browser.close()
        return pdf
```

**Result:** Pixel-perfect reports with charts.

## Scaling to 100K+ Requests/Month

### Database Optimization

**Problem:** N+1 queries killing performance.

```python
# ‚ùå Bad (N+1 queries)
expenses = Expense.objects.filter(profile=profile)
for expense in expenses:
    print(expense.category.name)  # Query on every iteration!

# ‚úÖ Good (1 query)
expenses = Expense.objects.filter(profile=profile).select_related('category')
for expense in expenses:
    print(expense.category.name)  # No extra query
```

**Impact:** 200ms ‚Üí 20ms for expense list.

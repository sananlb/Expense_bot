### 4. Создать утилиту `bot/utils/language.py`
**Проблема:** Дублирование кода получения языка в 3 местах
**Файлы затронуты:**
- `bot/services/expense_functions.py:131, 233, 1508`

**Действие:** Создать новый файл `bot/utils/language.py`:
```python
"""Утилиты для работы с языками пользователей"""
import logging
from typing import Optional
from expenses.models import Profile

logger = logging.getLogger(__name__)

def get_user_language_safe(user_id: int, default: str = 'ru') -> str:
    """
    Безопасно получить язык пользователя с fallback.

    Args:
        user_id: Telegram ID пользователя
        default: Язык по умолчанию (ru/en)

    Returns:
        Код языка пользователя или default
    """
    try:
        profile = Profile.objects.get(telegram_id=user_id)
        return profile.language_code or default
    except Profile.DoesNotExist:
        logger.debug(f"Profile not found for user {user_id}, using default: {default}")
        return default
    except Exception as e:
        logger.warning(f"Error getting language for user {user_id}: {e}")
        return default
```

**Затем заменить в expense_functions.py:**
```python
from bot.utils.language import get_user_language_safe

# Вместо:
try:
    profile = Profile.objects.get(telegram_id=user_id)
    lang = profile.language_code or 'ru'
except:
    pass

# Использовать:
lang = get_user_language_safe(user_id)
```

**Приоритет:** P1
**Время:** 30 минут (создать + заменить в 3 местах)

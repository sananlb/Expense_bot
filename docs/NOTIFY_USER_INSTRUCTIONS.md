## Шаги

### 1. Обновить код на сервере
```bash
ssh batman@176.124.218.53
cd /home/batman/expense_bot && bash scripts/quick_bot_update.sh
```

### 2. Проверить что баг исправлен
- Зайти в бота под тестовым аккаунтом
- Попробовать создать категорию дохода
- Убедиться что ошибки нет

### 3. Отправить уведомление пользователю
```bash
ssh batman@176.124.218.53
cd /home/batman/expense_bot
docker-compose exec -T bot python3 notify_user_bug_fixed.py
```
